# Licencia Creative Commons

Este proyecto academico implementa la licencia Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0) para los contenidos no ejecutables creados para la aplicacion, como textos, documentacion, capturas, guias, mockups y materiales de presentacion.

## Como se implementa

- Se agrega el archivo `NOTICE-CC.md` como aviso visible dentro de `app-android`.
- Se declaran `cc_license_name` y `cc_license_url` en `app/src/main/res/values/strings.xml`.
- La licencia permite compartir y adaptar los materiales, siempre que se atribuya al equipo y las obras derivadas se compartan con la misma licencia.

## Atribucion sugerida

"Gestion Eventos Comunitarios", por el equipo del proyecto, bajo Creative Commons Attribution-ShareAlike 4.0 International. Licencia: https://creativecommons.org/licenses/by-sa/4.0/

## Alcance

El codigo fuente puede mantenerse bajo la licencia del repositorio principal. Esta licencia Creative Commons se aplica a los contenidos y recursos academicos no ejecutables del proyecto.
